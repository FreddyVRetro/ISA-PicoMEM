I. Introduction
---------------

   This diskette contains the latest CD-Rom driver version 4.19
   for DOS supporting Creative CD-ROM Model CR521/523/563 drives.
   
   Also solve "NOT READY" error when reading "HP Multimedia Demo"
   CD Title.
